## 핵심 기능 3가지

### 기능 1. AI 카피 생성 (Generate)

내 상품/매장을 설명하면 AI가 마케팅 카피 3개를 자동으로 만들어줍니다.

**사용 방법:**

1. 왼쪽 메뉴에서 **"Generate"** 클릭
2. **상품/서비스 설명** 입력
   - 예: `고기리 막국수 - 용인 막국수 맛집, 고소한 막국수`
   - 팁: 자세히 쓸수록 카피 품질이 올라갑니다
3. **타겟 고객** 입력
   - 예: `30대~60대 가족외식`
4. **채널 선택** — 어디에 올릴 카피인지 선택
   - **인스타그램**: 짧은 카피 + 해시태그
   - **블로그**: 풍부한 본문 + SEO
   - **문자/SMS**: 40자 이내 간결
   - **네이버 플레이스**: 방문 유도 리뷰
5. **카피 톤** 선택
   - **친근**: 따뜻하고 편안한 말투
   - **전문**: 신뢰감 있는 말투
   - **유머**: 재치 있는 말투
   - **하드셀**: 강력한 구매 유도
6. **"카피 생성하기"** 클릭 → AI가 카피 3개를 만들어줍니다

**결과 화면에서 할 수 있는 것:**

- 카피 1 / 카피 2 / 카피 3 비교하기
- 마음에 드는 카피 **수정** (제목, 본문, CTA, 해시태그 각각 편집 가능)
- **복사** 버튼으로 SNS에 바로 붙여넣기
- **저장** 버튼으로 히스토리에 보관
- **카드뉴스 만들기** 버튼으로 이미지 제작

### 기능 2. 리뷰 분석 (Upload)

고객 리뷰를 AI가 분석해서 마케팅 인사이트를 뽑아줍니다.
분석 결과를 바로 카피 생성에 연결할 수 있어 더 정확한 카피를 만들 수 있습니다.

#### 2-1. 엑셀 리뷰 분석

네이버, 카카오맵, 배달앱 등에서 수집한 리뷰를 엑셀로 분석합니다.

**사용 방법:**

1. 왼쪽 메뉴에서 **"Upload"** 클릭
2. **"엑셀 분석"** 탭 선택
3. 엑셀 파일(.xlsx) 업로드
   - 필수 컬럼: `content` (또는 `리뷰내용`)
   - 선택 컬럼: `rating`, `date`, `platform`
4. **"AI 분석 시작"** 클릭

**분석 결과:**
- 긍정 키워드 (고객이 좋아하는 점)
- 부정 키워드 (개선이 필요한 점)
- 추천 타겟 고객층
- 추천 카피 톤
- 핵심 강점 / 약점
- 카피 힌트 (이걸 강조하면 효과적!)

5. **"이 분석으로 카피 만들기 →"** 클릭하면 분석 결과가 자동으로 카피 생성 페이지에 채워집니다

**엑셀 파일 준비 팁:**
```
| content              | rating | platform |
|----------------------|--------|----------|
| 막국수가 정말 맛있어요 | 5      | 네이버   |
| 양이 좀 적어요        | 3      | 카카오맵 |
```

#### 2-2. 이미지 리뷰 분석

고객이 올린 리뷰 사진(음식, 제품, 매장 사진)으로 고객 선호도를 분석합니다.

**사용 방법:**

1. **"이미지 분석"** 탭 선택
2. 리뷰 이미지 업로드 (최대 50장, jpg/png/webp/gif)
   - 여러 번 클릭해서 이미지를 추가할 수 있습니다
   - 잘못 올린 이미지는 X 버튼으로 삭제
3. **"이미지 AI 분석 시작"** 클릭

**분석 결과:**
- 인기 항목 (고객이 가장 많이 찍은 메뉴/제품)
- 고객 시각적 선호 (어떤 장면을 찍는지)
- 이미지별 상세 분석
- 마케팅 인사이트 (타겟, 톤, 강점, 카피 힌트)

4. **"이 분석으로 카피 만들기 →"** 클릭하면 카피 생성으로 연결

**활용 팁:**
- 네이버 플레이스, 인스타그램, 카카오맵 리뷰에서 고객이 올린 사진을 저장해서 업로드하세요
- 10장보다 30~50장을 분석하면 더 정확한 선호도 패턴을 파악할 수 있습니다

#### 2-3. 분석 히스토리

**"분석 히스토리"** 탭에서 과거 분석 결과를 다시 확인하고 재사용할 수 있습니다.

### 기능 3. 카드뉴스 만들기 (Card News)

생성된 카피를 인스타그램용 이미지(1080x1080)로 바로 만들 수 있습니다.

**사용 방법:**

1. 카피 생성 결과 화면에서 **"카드뉴스 만들기"** 클릭
2. **카피 선택**: 카피 1 / 카피 2 / 카피 3 중 선택
3. **테마 선택**: 5가지 디자인 테마
   - Dark (어두운 배경)
   - Light (밝은 배경)
   - Gradient (그라데이션)
   - Mint (민트색)
   - Orange (오렌지색)
4. 미리보기에서 디자인 확인
5. **"PNG 다운로드"** 클릭 → 바로 인스타에 올릴 수 있는 이미지 저장

## 추가 기능

### 대시보드 (Dashboard)

로그인 후 대시보드에서 내 활동을 한눈에 확인할 수 있습니다.

- 총 카피 생성 횟수
- 총 리뷰 분석 횟수
- 채널별 생성 현황 (인스타/블로그/SMS/네이버)
- 최근 카피 생성 내역
- 최근 리뷰 분석 결과

### 히스토리 (History)

과거에 생성한 모든 카피를 보관합니다.

- **검색**: 제목으로 검색
- **채널 필터**: 인스타/블로그/SMS/네이버별로 필터링
- **재사용**: "다시 생성" 버튼으로 같은 조건으로 새 카피 생성
- **복사**: 카피를 바로 클립보드에 복사
- **삭제**: 불필요한 히스토리 정리

## 추천 활용 시나리오

### 시나리오 1: 식당 사장님

```
1. 네이버 플레이스에서 내 가게 리뷰 엑셀 다운로드
2. Upload → 엑셀 분석 → 고객이 뭘 좋아하는지 파악
3. "이 분석으로 카피 만들기" 클릭
4. 인스타그램 채널 + 친근한 톤 선택
5. 카피 3개 생성 → 마음에 드는 거 선택
6. 카드뉴스 만들기 → PNG 다운로드
7. 인스타그램에 바로 업로드!
```

### 시나리오 2: 온라인 쇼핑몰 운영자

```
1. 상품 리뷰 사진 50장 모아서 이미지 분석
2. 고객이 어떤 장면을 좋아하는지 파악
3. 분석 결과로 카피 자동 생성
4. 블로그 채널 선택 → SEO 최적화 카피 생성
5. 블로그 포스팅에 활용
```

### 시나리오 3: 마케팅 대행사

```
1. 클라이언트 리뷰 데이터 분석 (엑셀 + 이미지)
2. 4개 채널(인스타/블로그/SMS/네이버)별 카피 각각 생성
3. 히스토리에서 모든 카피 관리
4. 카드뉴스까지 한 번에 제작
5. 클라이언트에게 제안서와 함께 전달
```

## 자주 묻는 질문 (FAQ)

**Q. 카피 생성은 무료인가요?**
A. 현재 테스트 기간에는 무료로 이용 가능합니다.

**Q. 엑셀 파일 형식은 어떻게 만드나요?**
A. 첫 번째 행에 컬럼명(`content` 또는 `리뷰내용`)을 적고, 아래에 리뷰 내용을 한 줄씩 입력하세요. .xlsx 형식으로 저장하면 됩니다.

**Q. 이미지 분석은 몇 장이 적당한가요?**
A. 최소 10장 이상, 30~50장이면 신뢰도 높은 분석 결과를 얻을 수 있습니다.

**Q. 생성된 카피를 수정할 수 있나요?**
A. 네. 결과 화면에서 제목, 본문, CTA, 해시태그를 각각 편집할 수 있습니다.

**Q. 카드뉴스 이미지 크기는 얼마인가요?**
A. 1080x1080px로 인스타그램 피드에 최적화되어 있습니다.
